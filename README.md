# 📊 Data Science Job Market Analyzer

An interactive web app to analyze data science salaries worldwide. Compare roles, explore location trends, and predict your salary using machine learning.

## ✨ Features

- **Role Comparison** – Compare salaries across job titles and experience levels
- **Location Analysis** – Interactive world map of salaries by country
- **Salary Predictor** – ML model predicts your expected salary
- **Insights Dashboard** – Key trends and statistics at a glance

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas / NumPy
- Plotly / Seaborn
- Scikit-learn

## 📁 Dataset

Source: Data Science Job Salaries 2024 (Kaggle)
- 14,000+ records
- 2020–2024 data
- Global coverage
